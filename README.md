# StudentInformationSystem_Nacua_Kinsley

This project is a simple Student Information System developed using Java (Swing), MySQL, and JDBC. It allows users to manage student records through a graphical interface.

Features:

• Add student
• View students
• Update student
• Delete student

Technologies Used:

• Java (NetBeans)
• MySQL
• JDBC

Database:

• Database: student_information_system
• Table: students

How to Run:

1. Open the project in NetBeans
2. Make sure MySQL is running
3. Run the application
4. Use the GUI to manage students
